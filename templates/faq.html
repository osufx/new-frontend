What is osu!fx?

It is a custom client for the very popular game osu! (http://osu.ppy.sh)

How do I install osu!fx?

At the moment you need to unzip the custom files into a working osu! game folder. I will add a feature soon to make it easier to install.

How do I go back to normal osu? (Uninstall)

Just use osu's installer to reinstall osu! If you point the installer to install into your current osu! folder it will install osu! there and you will keep all your settings and maps. I will make it possible to install osu!fx in a seperate folder soon. (It is possible but a bit tricky)
